# Xotosphere

Web app built by Node.js and Vue.js

## Backend

Url: http://localhost:5000

```
cd backend
```

### Install dependencies

```
npm install
```

### Compile code

```
npm run build
```

### Run Server

```
npm run serve
```

## Frontend

Url: http://localhost:8080

```
cd frontend
```

### Install dependencies

```
npm install
```

### Run server

```
npm run serve
```

### Build for production

```
npm run build
```
